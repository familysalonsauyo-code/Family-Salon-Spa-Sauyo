# Family Salon & Spa Sauyo

A professional website for a family salon and spa. Showcasing services, pricing, gallery, and more.

## Getting Started

### Setup

Open any of the HTML files in your browser:
- `index.html` - Home page
- `gallery.html` - Service gallery
- `pricing.html` - Pricing information
- `services.html` - Service details
- `about.html` - About the salon
